# 🕌 تطبيق المسجد — بوابة الترجمة الفورية للخطبة

تطبيق ويب خفيف (بدون أي إطار عمل) يربط بين **منبر الإمام** و**هواتف المصلين** لترجمة الخطبة فوريًا لغير الناطقين بالعربية، مع كشف تلقائي للآيات القرآنية والأحاديث النبوية المذكورة في الخطبة وعرض ترجماتها المعتمدة.

## الملفات

| الملف | الوصف |
|---|---|
| `imam.html` | لوحة تحكم الإمام: نشر نص الخطبة كتابةً، أو بث صوتي مباشر يتحول لنص تلقائيًا (Web Speech API) ويُرسل إلى Firebase |
| `musalli.html` | بوابة المصلّي: ترجمة فورية للخطبة (إنجليزي / أوردو / بنغالي / إندونيسي)، كشف الآيات والأحاديث، بحث محلي في القرآن والسنة، وضع ليلي وتحكم بحجم الخط |
| `quran-data.json` | قاعدة بيانات محلية: 6,236 آية بالنص العربي + ترجمة إنجليزية معتمدة |
| `hadith-data.json` | قاعدة بيانات محلية: ~14,700 حديث من صحيح البخاري ومسلم مع ترجمات إنجليزية |
| `worker/translate-worker.js` | بروكسي ترجمة موثوق (Cloudflare Worker → Azure Translator) يحمي المفتاح السري |
| `worker/README_WORKER.md` | دليل إعداد خدمة الترجمة الموثوقة خطوة بخطوة (مجاني بالكامل) |

## المعمارية

```
🎙️ الإمام (imam.html)
   │  نص الخطبة (كتابة أو تعرف صوتي ar-SA)
   ▼
☁️ Firebase Realtime Database  (مسار sermon)
   │  استعلام دوري كل 0.7 ثانية
   ▼
📱 المصلّي (musalli.html)
   ├─ الترجمة الفورية:
   │    1) Cloudflare Worker → Azure Translator  (موثوق — يُفعَّل بضبط TRANSLATE_WORKER_URL)
   │    2) Google Translate غير الرسمي           (احتياطي)
   │    3) MyMemory                              (احتياطي أخير)
   └─ كشف الآيات والأحاديث: بحث محلي 100% في quran-data.json و hadith-data.json (بدون إنترنت)
```

## الإعداد السريع

1. **Firebase**: أنشئ مشروع Firebase وفعّل Realtime Database، ثم ضع رابط قاعدة البيانات في `FIREBASE_CONFIG` داخل **كلٍّ من** `imam.html` و `musalli.html` (يجب أن تكون القيم متطابقة تمامًا).
2. **الترجمة الموثوقة (اختياري لكن يُنصح به)**: اتبع دليل [`worker/README_WORKER.md`](worker/README_WORKER.md) لإنشاء مفتاح Azure Translator مجاني (2 مليون حرف شهريًا) ونشر الـ Worker على Cloudflare، ثم ضع رابطه في `TRANSLATE_WORKER_URL` داخل `musalli.html`. لو تركته فارغًا يعمل التطبيق بالخدمات المجانية الاحتياطية كما هو.
3. **الاستضافة**: ارفع الملفات على GitHub Pages (أو أي استضافة ثابتة). افتح `imam.html` على جهاز المنبر و `musalli.html` على هواتف المصلين.

## ملاحظات أمان مهمة

- قاعدة Firebase الحالية تعمل بقواعد مفتوحة للتجربة — قبل الاستخدام الفعلي في مسجد حقيقي، فعّل قواعد أمان تمنع الكتابة إلا من الإمام (راجع قسم Rules في Firebase Console).
- مفتاح Azure لا يظهر أبدًا في كود المتصفح؛ يبقى محفوظًا كسر (Secret) داخل Cloudflare Worker.
